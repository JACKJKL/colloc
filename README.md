# colloc
一款烧脑的益智游戏。同名微信小游戏已上线。

开源出来作为`Pixi.js`游戏开发的入门教程。


### 开发

```bash
# clone 项目
git clone git@github.com:JetLua/colloc.git

# clone 子项目
git submodule init
git submodule update

# 安装依赖
npm i

# 运行
npm run dev

# 微信开发者工具选择dist目录
```
